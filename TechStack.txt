Backend & Data Processing

  Core

  - Python 3.x - Primary programming language
  - MySQL - Relational database (with ONLY_FULL_GROUP_BY mode)
  - Virtual Environment (.venv) - Python dependency isolation

  Web Scraping

  - BeautifulSoup4 4.14.2 - HTML parsing
  - Playwright 1.55.0 - Browser automation for JavaScript-heavy pages
  - Requests 2.32.5 - HTTP client
  - lxml 6.0.2 - XML/HTML parsing

  Data Processing

  - Pandas 2.3.3 - Data manipulation and CSV handling
  - NumPy 2.3.3 - Numerical computing

  Database

  - mysql-connector-python 9.4.0 - MySQL driver
  - SQLAlchemy 2.0.44 - SQL toolkit and ORM

  AI/RAG System

  LLM Integration

  - Anthropic Claude API (claude-sonnet-4-20250514) - Primary LLM
  - anthropic 0.69.0 - Official Python SDK
  - OpenAI 2.3.0 - OpenAI SDK (optional integration)

  RAG Framework

  - LangChain 0.3.27 - RAG orchestration framework
  - LangChain-Community 0.3.31 - Community integrations
  - LangChain-Core 0.3.79 - Core abstractions
  - LangChain-Text-Splitters 0.3.11 - Text chunking
  - LangSmith 0.4.34 - LangChain observability
